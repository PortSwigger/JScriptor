With this extension, you can import JavaScript code and it will be run like Pre-script and Post-script in Postman. You can also see the output of pre-script/post-script in the output of the extension..

Settings library

  • User can import JavaScript Library to extension in Setting Library Tab
  • Example: Add CryptoJS to Library
    • Download file cryptojs library
    • Import to extension
    • Use it

Further information on Global Objects, and examples can be found in the GitHub repository.
